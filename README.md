# MERN Stack Internship -- Task 1

This project is a simple MERN (MongoDB, Express, React, Node.js)
application built as part of an internship task.

The application demonstrates basic full-stack functionality where a
React frontend fetches data from an Express backend connected to MongoDB
and displays it on the UI.

------------------------------------------------------------------------

## Features

-   Node.js + Express backend API
-   MongoDB database connection using Mongoose
-   React frontend 
-   Fetch data from backend and render on UI
-   Basic full-stack integration

------------------------------------------------------------------------

## Tech Stack

Frontend: - React 

Backend: - Node.js - Express.js

Database: - MongoDB Atlas

------------------------------------------------------------------------


## Setup Instructions

### Backend

cd backend npm install node index.js

Server: http://localhost:5000

Seed data (run once): http://localhost:5000/seed

API: http://localhost:5000/api/users

------------------------------------------------------------------------

### Frontend

cd frontend npm install npm run dev

Frontend: http://localhost:5173

------------------------------------------------------------------------
